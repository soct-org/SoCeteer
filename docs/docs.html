SoCeteer Documentation

A framework for designing and running RISC-V-based SoCs on FPGA and in simulation, built on top of Chisel. Everything on this site is local to the repository — no network access required.

Getting started

Quick Start (README)

Generate a design, run it in Verilator, and build a ZCU104 bitstream with a custom DIMM — the fastest path from clone to running SoC.

Guides

Architecture & Hardware Flow

How a design goes from Chisel configs through Diplomacy, FIRRTL and firtool to Verilog, and on to Verilator or Vivado.

Bare-Metal Programs & soctglue

The runtime library, linker layout, heap and stack budgets, syscall handler chain, FatFs, and how to add programs and boot ROMs.

FPGA Memory & Custom DDR4

Selecting DIMMs with --ext-mem-part, the part registry, and how the custom (non board-flow) DDR4 interface works.
